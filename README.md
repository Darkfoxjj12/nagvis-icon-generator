# NagVis Icon Generator

A simple, fast, and 100% client-side web application to automatically generate a complete NagVis icon set (11 states) from a single transparent PNG image.

🌐 **Live Website:** [darkfoxjj12.github.io/nagvis-icon-generator](https://darkfoxjj12.github.io/nagvis-icon-generator/)

## 🚀 Features

* **Zero Backend:** All image processing is done locally in your browser using the HTML5 Canvas API. Perfectly secure.
* **100% Offline Capable:** Built as a single, standalone HTML file with zero external dependencies.
* **11 Native States:** Generates all standard NagVis states (`_up`, `_ok`, `_warning`, `_down`, `_critical`, etc.).
* **Smart Resizing:** Choose your icon size (8px to 128px). The aspect ratio of your original image is automatically preserved.
* **Customization:** Override default status colors if needed.
* **Batch Export:** Download everything in a single `.zip` file generated directly in the browser.
* **User Friendly:** Drag & drop support, Dark mode, and Bilingual (English/French).

## 🛠️ Usage

### Option 1: Use Online
Simply visit the [Live Website](https://darkfoxjj12.github.io/nagvis-icon-generator/), drop your PNG, and click "Generate".

### Option 2: Use Offline (Field operations)
No installation or build process is required. 
1. Download the `index.html` file or the latest release.
2. Open `index.html` in any web browser. 
3. The tool works entirely offline without any internet connection.
